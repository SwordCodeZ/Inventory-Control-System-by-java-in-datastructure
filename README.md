# 📦 Inventory Control System – Java + Data Structures

A beginner-friendly Inventory Control System built entirely in **Java** using **Swing** for the GUI and a simple **Binary Tree** data structure to manage inventory. This project includes features like login protection, low stock alerts, item search, bill generation, and file-based persistence to retain data across sessions.

---

## 🚀 Features

- 🧮 Binary Tree–based inventory structure
- 🔐 Login required to add stock
- 📊 View, add, and sell items via simple GUI
- 📉 Low stock warning (e.g., quantity < 200)
- 🧾 Multi-item billing system
- 💾 Data saved in `inventory_data.txt` and reloaded on startup
- 📜 History logging of stock operations
- 🖼 Branded GUI with logo, icons, and themed buttons
